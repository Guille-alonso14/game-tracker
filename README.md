# Game Tracker

Aplicación web para gestionar tu lista de videojuegos pendientes,
en progreso y completados.

## Tablero Trello

[Ver tablero de gestión del proyecto](TU_ENLACE_TRELLO)

## Tech stack

- Frontend: React + TypeScript + Tailwind CSS
- Backend: Node.js + Express
- Routing: React Router

## Instalación

\`\`\`bash
npm install
npm run dev
