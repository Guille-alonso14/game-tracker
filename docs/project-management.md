# Gestión del proyecto

## Metodología

Se ha aplicado Scrum de forma adaptada al proyecto individual:
- 4 sprints de un día cada uno
- Backlog definido al inicio con las 15 fases del ejercicio
- Entrega funcional al final de cada jornada

## Tablero Trello

El tablero se organiza en 5 columnas:
- **Backlog** — funcionalidades pendientes de planificar
- **Todo** — tareas planificadas para el sprint actual
- **In Progress** — tareas en desarrollo
- **Review** — tareas completadas pendientes de revisión
- **Done** — tareas finalizadas

## Organización del trabajo

Cada funcionalidad del producto se divide en subtareas técnicas
(componente, hook, endpoint, documentación) y se mueve por el tablero
según avanza el desarrollo.
